Apa itu Showcase Website?
Showcase Website adalah sebuah website yang digunakan untuk menampilkan Produk atau Jasa dengan tujuan untuk meningkatkan Brand Awarness
Apa itu Source Code?
Source Code merupakan sekumpulan Kode yang digunakan untuk membuat sebuah website
Apa itu Domain dan Hosting?
Domain merupakan nama/alamat dari sebuah website dan bersifat Unik, sedangkan Hosting merupakan layanan penyimpanan untuk menyimpan website agar dapat diakses secara Online
